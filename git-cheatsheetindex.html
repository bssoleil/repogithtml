Commandes Actions
git init créer un nouveau dépôt GIT
git statuss affiche la liste des fichiers modifiés
git add ajouter des fichiers à l'index
git commit stocke les changements de l’Index dans le dépôt
git diff Affiche les différences par rapport à l’Index
git log Affiche la liste des derniers commits
git push envoie les modifications locales apportées à la branche principale associée
git checkout pour créer des branches ou pour basculer entre elles
git checkout passer d’une branche à une autre
git remote permet à un utilisateur de se connecter à un dépôt distant
git remote -v répertorie les dépôts distants actuellement configurés
git branch pour répertorier, créer ou supprimer des branches
git branch –d Pour supprimer une branche
git pull Pour récupérer toutes les modifications présentes sur le dépôt distant
git merge utilisée pour fusionner une branche dans la branche active
git diff permet de lister les conflits
git diff --base Pour visualiser les conflits d’un fichier
git fetch permet d’extraire tous les fichiers du dépôt distant non présents dans le répertoire
git rebase master utilisée pour la réapplication des commits sur une autre branche
